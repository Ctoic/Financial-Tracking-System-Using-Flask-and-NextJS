Update Students via Excel

Instructions:

Please upload an Excel file (.xlsx or .xls) with the following columns:

  • name - Student's full name
  • fee - Fee amount (numeric)
  • room_id - Room number (1-8)
  • status (optional) - Student status (active/inactive/graduated)
  • picture (optional) - Filename of the student's picture (must exist in static/uploads folder if provided)

Important Notes:

  • For existing students: If a student with the same name exists, their information will be updated.
  • For new students: Only name, fee, and room_id are required. Picture and status are optional.
  • If you provide a picture filename, make sure the file exists in the static/uploads folder.
